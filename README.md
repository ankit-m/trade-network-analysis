World Trade Network Analysis
============================

### System Requirements

1. Python ^2.7.1
2. Python igraph
3. COW Dyadic Dataset
4. numpy*
5. matplotlib*

**optional*

Refer to websites for download instructions

### Setup

To setup your environment. Type

```shell
python setup.py
```

1. http://www.eief.it/files/2010/10/luca-de-benedictis.pdf
